# .vim

These are my opinionated .vim settings.  I clone it somewhere - [clonedir]

    $ cd [clonedir]
    $ ./setup.sh

## Uses submodules to manage plugins

    $ cd ~/.vim
    $ git submodule add <git-path-to-plugin> bundle/<plugin-name>
